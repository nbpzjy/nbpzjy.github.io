The biggest problem

AttributeError: 'Series' object has no attribute 'order'

See the book snapshot:

Do the exact same thing on the JupyterNotebook, but it continues give the AttributeError: ‘Series’ object has no attribute ‘order’.

Search on Google & Baidu, nothing found. Hard to believe no body got this error before. If any one try to type the code as per the book, this answers should comes out on the Google or Baidu.

Try a lot of methods, none worked.

Official web page Pandas, read the API, find that the latest version of Pandas was no longer support the .order function. To make the re-arranged date, there were only two ways: sort_index, sort_values.

Careful:

  1. sort_index:

    Series.sort_index(axis=0, level=None, ascending=True, inplace=False, kind='quicksort', na_position='last', sort_remaining=True

  2. sort_values:Here must be values

    Series.sort_values(axis=0, ascending=True, inplace=False, kind='quicksort', na_position='last')[source]
