Hexo重新开始啦

很久以前用过的HEXO，后来没怎么用了，iMac用起来一直想把博客继续，可是一直比较懒。

搞了半天，总感觉imac下网速实在太慢，下载东西感觉只有几k，蛋疼。甚至把怨气洒到了安装的其他程序上，比如istate，handoff等。感觉似乎也用不着。就卸载了。

iMac关机有花屏出现

截止2017-02-20，我总共已经遇到了三次关机花屏的问题，第一次是chrome在下载东西，阻止关机我强制退出，出现了第一次。

今天出现两次，安装一个叫mount ducker，据说是可以吧网盘如Googledrive，dropbox等挂到本地的一个app，安装完成并第一次挂载Dropbox的时候，失去连接之后就死了，鼠标滑上去是在转，并且向用cleanmymac来清理，导致cleanmymac也是鼠标滑上去就转圈。不得已关机，却是关机也关不了，等了很久之后出现花屏，关机了。

第三次则是卸载mount ducker之后，还是很不流畅，chrome无响应，finder也是，结束finder重新启动时，花屏之后进入黑屏。

解决了每次开机chrome自动打开的问题

系统偏好设置——用户与群组——选择当前用户——登录项——把里面的chrome删掉，妥了。

重新安装完成Hexo

安装hexo，之前先安装node.js

下载node.js半天也没下来，最好求助迅雷下载宝搞定。下载的是最新版的node.js。

Hexo安装时要用sudo npm hexo-cli -g，否则会因为权限问题失败。
同样在安装git方式部署，即 sudo npm hexo-deployer-git —save的时候前面也要sudo才行

安装完成即可预览了，由于拷贝了之前的hexo文件夹，重新安装之后除了主题变为默认之外内容都在。为了继续用之前的主题（懒得再设置），预览成功之后把hexo拷贝过去把所有文件覆盖。搞定。
