在 After Effects 导出视频，一个片头，很慢。现在对视频处理越来越没兴趣了，可是伙伴们却总有些活要干——说实在的，真心不想弄了，什么时候学会说“不”呢？

在等待的这半个多小时里，打开了 GitHub ， 浏览一下大牛们都在干嘛。无意间点开了一个 Facebook 开源的 litho 。 虽然不知道是什么，也打开来看看：万一不小心看懂了呢？于是就来到了 Facebook 的 Github 主页，于是就看到了一直听说而一直没有用过的 Fresco。

Fresco 是Facebook开源的图片加载库

事实上有很多的图片加载库，由于一直处在学与不学之间，所以很多东西都是听过却没用过。Fresco就是一个听说过而没用过的。在等待这段 After Effects的这段时间正好学习一下怎么用这个库。

其实蛮简单的。有一个中文的官方指导文档，介绍得还算详细——当然如果一点基础都没有可能看一些东西也会觉得莫名其妙。当然现在可以看我的这个小文章，包你会用。
