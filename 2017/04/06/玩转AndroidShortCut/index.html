有很长很长时间没有撸过Android的代码了。嘻嘻算起来应该是去年十一之后就没真正意义上敲过了。虽然偶尔在看视频，但是真的没动手去实现过任何东西。深深地为自己的行为感到惭愧。

看官方文档学习

第一次，跟着官方的文档去学个东西。AppShortCut(可能需要科学上网才能看到)。

严格说应该是上个月就敲完了这几行代码的，调试没成功。原因是我的虚拟设备是api24的，而Short cut只能在大于api25的设备才能运行。唯一的就是重新创建一个api25的虚拟机。遗憾的是当时SS的流量超标了。在这个月初由于某系原因，一直也没能成功升级虚拟机。一晃就到了6号。

功能介绍

这个功能在谷歌的原生app中有好几个已经实现了。在图标上长按，会弹出一个列表来，列表中有一些功能，可以再不用先运行app的情况下直接进入需要的功能。

比如常用的keep，长按弹出的列表：

  • New note
  • New list
  • New photo note
  • New audio note


可以在不运行keep的情况下点击相应的选项进入需要的功能。

这些弹出的列表功能，还可以按住该功能放到桌面——感觉有如另外一个app那样。
