写作，被逼的宠爱

又催我写文章，我就睡了两觉，做了个饭，洗了个碗，出去春游了一把。

非要我写，我没有感受哪有芬芳的文字吐露。

无事也写写，试试他教我的格式。这种依从，是不是对他最大的宠爱。

此处省去五个字，我现在真的懒得想，是个得过且过的人，超过三个月的事不想不提，只关心吃喝拉洒睡，要说上追求，无非就是把自己变得美美哒，萌萌哒，小时候想学的钢琴，跳舞都学一下，主要为了自己学了，好和将来要学的豆豆看齐。
