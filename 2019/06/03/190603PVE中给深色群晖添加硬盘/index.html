PVE 虚拟机下挂载硬盘

2019-06-03 | 网络技术
条评论

在PVE节点选择 Shell, 进入命令窗口

查看插入的硬盘
ls /dev/disk/by-id

fdisk -lu

格式化分区
fdisk /dev/sdc

可以输入 m 查看帮助

创建新的分区用 n 命令

选择创建主分区

一路默认下来，就创建成功了，分区表写入磁盘，输入 w

这样就完成创建分区了，并退出 fdisk

格式化分区
mkfs -t ext4 /dev/sdc1

创建目录用于挂载分区
mkdir -p /mnt/sdc1

挂载分区到目录
mount -t ext4 /dev/sdc1 /mnt/sdc1

查看挂载情况
df -h

开机磁盘自动挂载
echo /dev/sdc1 /mnt/sdc1 ext4 defaults 0 2 >> /etc/fstab

查看自动挂载配置
cat /etc/fstab

./img2kvm ds918+1.04b.img 103 vm-103-disk-1

PVE安装黑色群晖
winSCP 登录 PVE 根目录，上传 img2kvm，还有一个是群晖的引导 img

完毕之后关闭 winSCP，回到 PVE shell

PVE群晖添加挂载硬盘

先查看所有的硬盘
ls -l /dev/disk/by-id/

确定要挂载的硬盘id，copy下来
ata-ST4000VN008-2DR166_ZGY4XP24

ata-SAMSUNG_HM320JI_S16LJ90S701201

这个命令就直接挂载了
qm set 103 --sata2 /dev/disk/by-id/ata-ST4000VN008-2DR166_ZGY4XP24

qm set 105 --sata1 /dev/disk/by-id/ata-SAMSUNG_HM320JI_S16LJ90S701201

这里一定要注意是挂载给哪个虚拟机，编号一定要准确，同时要记得修改指定sata序号。
