我们的2017年

Apr 16, 2019 | 视频 | Hits

现在是2019年，2017年已经过去好久了，怎么现在才写个总结呢？因为现在才把hexo又重新搞定。

其实我的hexo一直没出问题，但是有点懒，又不知道该写点啥，一拖就是几个月半年的过去了。现在除了在 Github 托管之外还在 NAS 上跑起了web服务，虽然运营商封锁了 80/443 等端口，没法优雅地访问 web 服务，但是加个端口号再路由器端口转发还是可以搞定的。

2017年，过去一年多了，还是一个值得纪念的一年。

这一年，工作学习生活都算平安，孕育并生下了第二个儿子。上海学习，也算是过得蛮开心的，压力是有一些，总体来讲是舒服的。妻子也很给力，孕期坚持上班不说，每个周末都回家看望大儿子，辛苦是辛苦的，显然还算是比较充实。上海生下了第二个儿子，也算是为上海之行，画下浓墨重彩的一笔。下面是一些视频，就算是记录我们的2017吧。音乐是银临的《浮生辞》，古典风格，让人安静、沉思，为过去感怀，为将来祝福。

愿你们一切都好。
