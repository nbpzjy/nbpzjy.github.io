在 PVE 上部署 docker 环境

root@pve:~# apt-get install -y apt-transport-https ca-certificates wget software-properties-common

安装 curl 库

apt-get install curl

添加证书

curl -fsSL https://apt.dockerproject.org/gpg | apt-key add -qq

成功之后显示OK

添加docker库

root@pve:~# echo "deb https://apt.dockerproject.org/repo debian-stretch main" > /etc/apt/sources.list.d/docker.list

回车之后啥也看不见，但是可以通过 cat 命令查看是否添加成功

cat /etc/apt/sources.list.d/docker.list

得到的结果

deb https://apt.dockerproject.org/repo debian-stretch main

查看当前docker版本

apt-cache policy docker-engine

安装 1.12.6版本
apt-get install -y docker-engine=1.12.6-0~debian-stretch --allow-unauthenticated

部署管理界面

docker run -d -p 8007:9000 -v /var/run/docker.sock:/var/run/docker.sock \ --name ui --restart=always lihaixin/portainer:me

查看docker

docker ps

docker 信息

docker info

IP 配置表查看

root@pve:~# iptables -L -n -v
