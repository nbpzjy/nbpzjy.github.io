创建一个空的虚拟机

在节点下，创建一个新的虚拟机，磁盘不用任何介质
基本按默认来，一路下一步下一步，之后可以再修改。网络使用默认的半虚拟化。最后确认，完成

点击刚才创建的虚拟机。
磁盘需要删除：
分离——>删除。

至此，空的虚拟机完成。

下载并转换 LEDE 固件

koolshare 固件下载

连接：

在 PVE 节点，点击 shell，进入终端。

下载命令

wget

ls -al 可以查看是否有此文件

解压命令 gzip

gzip 带 -d 参数，要解压的文件名

gizp -d openwrt-koolshare-mod-v2.31-r10822-50aa0525d1-x86-64-combined-squashfs.img.gz

完成后再查看是否有 .img 的文件解压成功

转换格式命令

qemu-img -f raw -o qcow2 转换前文件名 转换后文件名

qemu-img -f raw -o qcow2 openwrt-koolshare-mod-v2.31-r10822-50aa0525d1-x86-64-combined-squashfs.img vm-152-disk-1.qcow2

转换后的文件命名：
vm+虚拟机的编号+磁盘+编号+文件名后缀，例如： vm-152-disk-1.qcow2

检查转换后的文件是否完整

qemu-img check 文件

查看转换后的文件信息

qemu-img info 文件

导入 PVE

local（pve）镜像文件
local-lvm(虚拟磁盘）
导入虚拟机的即是由 local 导入到 local-lvm
命令

qm importdisk xxxxx local-lvm

导入时注意名字不能错，不然会失败。

虚拟机添加磁盘

导入的磁盘会显示未使用的磁盘，双击->添加，挂载

WEB 界面点开机

开机之后，需要设置 IP 地址
开机成功，再点回车，查看ip

ifconfig | less

查看 IP 地址，之后 q 返回命令行

vi /etc/ipconfig/network

进入编辑界面，按 i 开始编辑
完成之后按 ESC 键，之后输入:wq 回车，退回命令行，再 reboot 重启
重启之后可以再查看 IP 是不是刚才设置的。

进入web 页面管理 LEDE

浏览器输入 IP 地址即可进入，默认密码 koolshare，然后就可以飞翔了。
